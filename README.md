# Todo List API - FlyRank Week 2 Assignment

A simple, lightweight CRUD API built with Node.js and Express to manage a personal to-do list. This project includes interactive API documentation built with Swagger UI.

---

## 🚀 Features

* **Health Check Endpoint:** Verify if the server is up and running.
* **Full CRUD Operations on Tasks:**
  * **Create:** Add new tasks with title validation.
  * **Read:** Retrieve all tasks or fetch a single task by its ID.
  * **Update:** Edit task titles or toggle their completion status (`done`).
  * **Delete:** Remove tasks from the list.
* **Interactive Documentation:** Beautiful UI to test all endpoints directly from your browser.

---

## 🛠️ Tech Stack

* **Runtime:** Node.js
* **Framework:** Express
* **Documentation:** Swagger UI (via `swagger-ui-express`)
